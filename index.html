putrama

rub public transport map

idea

a lot of students, when moving to a new city, are new to the neighborhood and might not know how long it actually takes to the university. so create a map that shows in overlayed colors how long it will take from a given location to the university by using the public transport system.

notes

the distance to the stop is not accurate, but is measured as the distance to the next
stop. routes may take different stops.

usage

for now everything is quite hardcoded. roughly you need to hack the rectangle of 'start'points (in latitute/longitude coordinates), add the date and hour to consider, and run the script. it will create a .kml file, with the, named like "map_coord_time.kml". this map you can now easily open in google earth, or use any other geo program that can overlay that to the map you want.

prerequisites for the script: Geo::KML::PolyMap.
